Getting Started
  1. Masters → Centers – Create centers where members meet.
  2. Masters → Members – Register members and assign to a center.
  3. Loans → Masters → Loan Types – Define loan products with interest rates.
  4. Loans → Loan Applications – Submit a loan request for a member.
  5. Loans → Loan Approvals – Admin approves or rejects applications.
  6. Loans → Loan Disbursement – Admin disburses the approved amount.
  7. Loans → Posting → Recovery Posting – Post weekly/monthly repayments.
